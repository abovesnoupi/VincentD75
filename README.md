## Installation: 
```
# Clone project
git clone https://github.com/abovesnoupi/VincentD75.git

cd VincentD75

npm install             # Install dependencies
npm start               # Dev server + live reload
npm run start:server    # Start websockets server
```

```
npm run build   # Build project
```
